To access the application:

{{- if .Values.ingress.enabled }}

1. Ingress is enabled. Add this entry to your /etc/hosts file:

   {{- if eq .Values.ingress.className "nginx" }}
   # For Minikube:
   $(minikube ip) {{ .Values.ingress.host }}

   # For Kind:
   127.0.0.1 {{ .Values.ingress.host }}
   {{- else }}
   <cluster-ip> {{ .Values.ingress.host }}
   {{- end }}

2. Access the application at:
   http://{{ .Values.ingress.host }}

{{- else }}

1. Port forward to the front-ui service:
   kubectl port-forward -n {{ .Values.global.namespace }} svc/{{ .Release.Name }}-front-ui {{ index .Values "front-ui" "service" "port" }}:{{ index .Values "front-ui" "service" "port" }}

2. Access the application at:
   http://localhost:{{ index .Values "front-ui" "service" "port" }}

{{- end }}
